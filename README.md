# Algoritmo para el cálculo de áreas, perímetros y volúmenes

## Descripción

Este proyecto presenta el diseño e implementación de una herramienta computacional para calcular el área, perímetro y volumen de diversas figuras geométricas planas y sólidas. Desarrollado con HTML, CSS, Canvas y JavaScript, integra funciones matemáticas y representación gráfica para ofrecer una experiencia interactiva y educativa.

La interfaz permite seleccionar figuras geométricas, ingresar sus dimensiones, validar datos ingresados de forma dinámica, calcular las propiedades geométricas y mostrar tanto los resultados como un dibujo correspondiente en un lienzo.

## Funcionalidades principales

- Cálculo de área y perímetro para figuras 2D (cuadrado, rectángulo, círculo).
- Cálculo de volumen para sólidos (cubo, paralelepípedo, esfera, cilindro).
- Generación dinámica y validación de campos de entrada según la figura seleccionada.
- Dibujo visual de las figuras y sólidos en un canvas HTML.
- Resultados presentados de forma clara y responsiva para facilitar la comprensión.

## Tecnologías usadas

- HTML para estructurar la página.
- CSS para el diseño y estilo responsive.
- JavaScript para la lógica, cálculos, validación y generación dinámica de formularios.
- Canvas API para las representaciones gráficas.

## Instrucciones para uso

1. Clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador moderno.
3. Seleccionar la figura geométrica deseada.
4. Ingresar los valores de las dimensiones (lado, base, radio, etc.).
5. Presionar el botón “Calcular” para obtener resultados y ver la figura dibujada.
